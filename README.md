# Alpha Ichimatsu (アルファ・イチマツ)

VS Codeで画像を表示する際、透明部分 (アルファチャネル) を市松模様で表示する拡張機能です。
PNG, SVG, WEBP などの画像ファイルに対応しています。

## 機能

- **市松模様の背景**: 画像の透明部分が白と灰色の市松模様で表示されるため、透明部分を容易に識別できます。
- **対応フォーマット**: .png, .jpg, .jpeg, .svg, .gif, .webp

## 使い方

1. VS Codeで画像ファイルを開きます。
2. もしデフォルトで表示されない場合は、ファイルを右クリックして「次で開く... (Open With...)」を選択し、「Ichimatsu Image Viewer」を選択してください。
3. エディタのタブとして画像が開き、背景が市松模様で表示されます。

## 設定

現在、特別な設定はありません。

## 開発

```bash
# 依存関係のインストール
npm install

# コンパイル
npm run compile

# デバッグ実行
F5キーを押してデバッグを開始
```
